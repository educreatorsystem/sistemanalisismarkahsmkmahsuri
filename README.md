# Sistem Analisis Markah - SMK Mahsuri

Fail ini disediakan untuk publish melalui GitHub Pages.

## Kandungan
- `index.html` - fail utama sistem
- `kod_html_smk_mahsuri.txt` - salinan kod HTML untuk rujukan

## Cara Publish GitHub Pages
1. Buka repository GitHub.
2. Upload fail `index.html` ke root repository.
3. Pergi ke **Settings > Pages**.
4. Pilih **Deploy from a branch**.
5. Pilih branch `main` dan folder `/root`.
6. Klik **Save**.
